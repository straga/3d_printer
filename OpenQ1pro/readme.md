# BTT Eddy Duo for QIDI Q1 Pro

## BTT Eddy Duo functions:

This setup combines the functionality of homing, probing, and auto Z offset into one device.

- Home
- Probe
- Auto Z offset

## Installation

1. Install OpenQ1 on new NAND memory
   - Follow [OpenQ1 installation guide](https://github.com/frap129/OpenQ1/blob/main/docs/Installation.md)
   - Update all components

2. Install Eddy
   - Follow [BTT Eddy installation instructions](https://github.com/bigtreetech/Eddy)
   - For QIDI Q1 Pro:
     - Move WiFi module to upper USB port
     - Install Eddy on lower USB port (previously used for WiFi)
   - Use cable channel for USB cable routing

3. Physical Eddy installation
   - Set nozzle to Z0 using paper method
   - Place SD card between Eddy and print bed for initial spacing
   - Secure Eddy with screws

4. Calibration
   - Use a 20mm calibration object
   - Follow Eddy calibration instructions.
